# Joshua Storm Caley

Research software and platform engineer working in scientific computing, HPC, ML workflows, and structural biology infrastructure at UNSW.

I build reproducible workflows, researcher-facing platforms, and containerised software environments for biomolecular modelling and protein structure prediction workloads.

## What I work on

- Research software and workflow engineering
- Nextflow pipelines and reproducible scientific computing
- Open OnDemand applications for HPC users
- Containerised ML and structural biology tooling
- GPU-enabled workflows and platform support
- Open-source contributions in the scientific software ecosystem

## Featured work

### Creator and maintainer: Open OnDemand app for protein design
[`jscgh/ood-proteindesign`](https://github.com/jscgh/ood-proteindesign)

Open OnDemand app for protein design workflows, integrating form-driven submission with underlying Nextflow pipelines and institution-specific runtime configuration.

### Maintainer: nf-core/proteinfold
[`nf-core/proteinfold`](https://github.com/nf-core/proteinfold)

Contributor to the community-maintained protein structure prediction pipeline. Public contributions include support for additional prediction modes, image and dependency updates, workflow fixes, and HPC/runtime improvements.

### Core maintainer: Open OnDemand app for protein structure prediction
[`Australian-Structural-Biology-Computing/ood-proteinfold`](https://github.com/Australian-Structural-Biology-Computing/ood-proteinfold)

User-facing Open OnDemand application for running `nf-core/proteinfold` on HPC, including CI/CD, input handling, debugging improvements, and interface refinements.

## Selected public contribution highlights

- Added and improved support around `HelixFold3`, `RoseTTAFold-All-Atom`, and `Boltz` workflows in `nf-core/proteinfold`
- Built and maintained Open OnDemand apps that make advanced structural biology workflows more accessible to researchers
- Built reproducible tooling and configuration for multi-environment HPC execution, including systems such as NCI Gadi and Pawsey Setonix
